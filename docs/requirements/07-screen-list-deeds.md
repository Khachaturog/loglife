# 07. Экран: список дел

## Назначение

Главный экран (или один из основных), на котором пользователь видит все свои дела в виде карточек и может перейти к созданию, редактированию, заполнению формы или просмотру истории.

## Элементы экрана

- **Список карточек дел** — каждая карточка отображает эмодзи/символ, название, при необходимости краткое описание или превью; под названием — **«N сегодня · M всего»**: для дел только с «Да/Нет», текстом, выбором и т.п. (без числа/шкалы/duration) это **количество записей** за сегодня и всего; если в деле есть числовой блок — по правилам суммы или количества записей в зависимости от состава блоков.
- **Действия с карточкой** — только два варианта при взаимодействии с карточкой дела:
  - **Переход в дело** — открытие страницы просмотра дела ([14-screen-deed-view.md](14-screen-deed-view.md)) (эмодзи, название, описание, история записей; оттуда доступны редактирование и удаление дела).
  - **Добавить запись** — обычно переход на экран заполнения формы ([09-screen-fill-form.md](09-screen-fill-form.md)). Если включено **быстрое добавление** и заданы дефолты для всех блоков ([08-screen-create-edit-deed.md](08-screen-create-edit-deed.md)), короткое нажатие «+» создаёт запись с дефолтами; удержание «+» — форма (как на [экране дела](14-screen-deed-view.md)). При ошибке быстрого добавления — сообщение под карточкой.
- **Создание нового дела** — кнопка или действие «+ Создать дело» / «+ Добавить дело», переход на экран создания/редактирования дела (новая форма).
- **Профиль** — переход на страницу профиля (авторизация, экспорт данных — см. [12-screen-profile.md](12-screen-profile.md)). На мобильных доступен также через нижнюю панель вкладок ([15-layout-navigation.md](15-layout-navigation.md)).

## Навигация

- Со списка дел пользователь может перейти на экраны: страница просмотра дела ([14-screen-deed-view.md](14-screen-deed-view.md)), создание/редактирование дела ([08-screen-create-edit-deed.md](08-screen-create-edit-deed.md)), заполнение формы ([09-screen-fill-form.md](09-screen-fill-form.md)), профиль ([12-screen-profile.md](12-screen-profile.md)). История записей отображается на странице просмотра дела.
- На мобильных устройствах список дел является одной из вкладок нижней панели («Дела»); переход к списку дел возможен по этой вкладке с любого экрана в контексте дел и записей.

## Загрузка и обратная связь

- Пока подгружаются данные для счётчиков (второй запрос после списка дел), **Skeleton** (Radix Themes) заменяет только **числа**; подписи «сегодня ·» и «всего» остаются обычным текстом, без ложных нулей в цифрах.
- При **быстром добавлении записи** с карточки: индикатор ожидания на кнопке «+» появляется не раньше чем через **300 ms** после нажатия; после появления крутится **не меньше 500 ms** (чтобы не мигать на быстрых ответах сервера).
