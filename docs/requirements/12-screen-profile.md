# 12. Экран: Профиль

## Назначение

Страница профиля пользователя: настройки, авторизация и экспорт данных. Доступна из основного интерфейса по иконке профиля в шапке; на мобильных — также через вкладку «Профиль» нижней панели вкладок ([15-layout-navigation.md](15-layout-navigation.md)).

## Элементы экрана

### Аккаунт и пароль

- Email пользователя (если есть).
- Смена пароля.

### Экспорт данных

- Экспорт дел и записей в CSV (весь период или выбранный диапазон дат).

### Ссылки внизу экрана

Внизу экрана — блок ссылок, выровненных по центру:

- **Условия использования** — маршрут `/terms` (публичная страница).
- **Политика конфиденциальности** — `/privacy`.
- **Поддержать автора** — заглушка `href="#"` (позже можно подставить реальный URL в коде).
- **Проект на GitHub** — если в окружении задан `VITE_GITHUB_REPO_URL`.

## Навигация

- Переход на экран профиля — с главного экрана (список дел), через шапку приложения или, на мобильных, через вкладку «Профиль» нижней панели вкладок.
- С профиля пользователь возвращается к списку дел (по вкладке «Дела» на мобильных или по ссылке/кнопке «Назад») или другим разделам.

## Статус функций

- Авторизация и экспорт CSV: реализовано.
- Импорт дела с записями из CSV: не реализовано (при необходимости — отдельная задача).
